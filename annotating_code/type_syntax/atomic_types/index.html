アトミック型

アトミック型は、Psalmで使用されるすべての型情報の基本的な構成要素です。複数のアトミック型は、ユニオン型やインターセクション型と組み合わせることができます。Psalmでは、docblock構文で多くの異なる種類のアトミック型を表現できます。

注意: 以下のリストの各型をクリックすると、すべてのアトミック型の詳細なドキュメントと使用例を表示できます。

  • スカラー型
    • bool
    • int
    • float
    • string
    • int-range<x, y>
    • int-mask<1, 2, 4>
    • int-mask-of<MyClass::CLASS_CONSTANT_*>
    • class-string と class-string<Foo>
    • trait-string
    • enum-string
    • callable-string
    • numeric-string
    • literal-string
    • literal-int
    • array-key
    • numeric
    • scalar
  • オブジェクト型
    • object
    • object{foo: string}
    • Exception, Foo\MyClass と Foo\MyClass<Bar>
    • Generator
  • 配列型
    • array<int, string>
    • non-empty-array
    • string[]
    • list & non-empty-list
    • list<string>
    • array{foo: int, bar: string} と list{int, string}
    • callable-array
  • 呼び出し可能型
  • 値型
    • null
    • true, false
    • 6, 7.0, "forty-two" と 'forty two'
    • Foo\Bar::MY_SCALAR_CONST
  • ユーティリティ型
    • (T is true ? string : bool)
    • key-of<T>
    • value-of<T>
    • properties-of<T>
    • class-string-map<T of Foo, T>
    • T[K]
    • 型エイリアス
    • 変数テンプレート
  • その他の型
    • iterable<TKey, TValue>
    • void
    • resource
    • closed-resource
  • トップ型とボトム型
    • mixed
    • never
