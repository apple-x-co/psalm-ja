# アサインメント・トゥ・ボイド

`void` を返す関数から代入する際に発せられる：

```php
<?php

function foo() : void {}
$a = foo();
```

## なぜこれが悪いのか

`void` を返す関数は、PHPでは`null` を返すものとして扱われますが(そのため、 これ単体では実行時エラーにはなりません)、`void` は、プログラミング言語ではより広範な概念であり、代入を目的としたものではありません。

## 修正方法

代入を完全に削除すればよいのです：

```php
<?php

function foo() : void {}
foo();
```
