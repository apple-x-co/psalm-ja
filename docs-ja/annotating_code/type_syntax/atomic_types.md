# アトミック型

アトミック型は、Psalmで使用されるすべての型情報の基本的な構成要素です。複数のアトミック型は、[ユニオン型](union_types.md)や[インターセクション型](intersection_types.md)と組み合わせることができます。Psalmでは、docblock構文で多くの異なる種類のアトミック型を表現できます。
